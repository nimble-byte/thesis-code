# Usage Hints

Bulk rename:

```shell
python ./codebook.py relabel ./codebooks/codebook_v2.csv --select-verb "VERB" --to "PATTERN_LABEL" [--apply]
python ./codebook.py relabel ./codebooks/codebook_v2.csv --select-labels "LABEL" --to "PATTERN_LABEL" [--apply]
```

```shell
python ./codebook.py relabel ./codebooks/codebook_v2.csv --select-labels
```
